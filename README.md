## Clone Project
```bash
    git clone https://github.com/eps364/moveit-next.git
```

## Running

First, run the development server:

```bash
yarn install
# or
npm install
```

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
